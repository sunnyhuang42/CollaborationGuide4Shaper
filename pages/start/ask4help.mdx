# 如果遇到疑难，怎样寻求支持？

探索路上，肯定会遇到难题挑战。

如果你自己折腾一阵，还是没解决，还请多多撬动外力支持。

以下是一些资源线索，供参考：

## 如果是人生战略管理实践疑难

这类疑难主要指和这里的工具方法使用相关的疑难，比如这些工具方法和流程，使用起来会卡住、不知道怎么运用来帮助自己等。欢迎随时抛到飞书当月群中异步交流，或带到例会集体答疑环节，现场和主理人交流。


## 如果是自己人生发展上的具体疑难

这类疑难主要指自己面临的特定领域的难题，比如想打造自己的产品服务，不知道该从哪个客群的需求入手；和家人长期关系拧巴，不知道怎么处理等。

如果面临这类疑难，推荐参考下述四步，快速产出一版疑难的解法，然后根据个人情况迭代完善解决方案：

这个问题清单原文见当月飞书群顶部导航栏，欢迎随时取用：）



### 2. **如何做案头研究？**

刚开始我们的解法大概率是不够靠谱的，需要获取更多有效信息来迭代完善。案头研究是最常用的方式。推荐：

1. **先快速检索对应领域的优质信息源**，比如：
    - 人生发展（工作生活）的各种难题：[得到 App](https://www.dedao.cn/)（可直接全文检索全站电子书、专栏、锦囊、用户笔记等内容）、心智工具箱 公众号、[人生发展咨询手记](https://mp.weixin.qq.com/mp/appmsgalbum?__biz=MzA3MzM0MjUyMQ==&action=getalbum&album_id=2779446765539409925&scene=173&from_msgid=2652153334&from_itemidx=1&count=3&nolastread=1#wechat_redirect)等
    - 健康问题：[默沙东诊疗手册](https://www.msdmanuals.cn/home)、腾讯医典App或小程序
    - 商业、创业：[YC 相关资源](https://www.ycombinator.com/)，得到 App、华与华公众号、图书，刘润公众号等
2. 或者**问一下人工智能助手 ChatGPT** 。如何提升结果质量，可参考 [ChatGPT 答案质量不尽人意？这四大常见原因看你中几个 ](https://sunnyhuang.net/cmty/tips-gpt)
3. **如果仍未解决，则请教对应领域专家**；朋友圈内没有，就去[在行](https://www.zaih.com/)上看看有哪些行家可能带来灵感



### 3. 如何找人交流、请教、求助？

一些思路供参考：

1. 梳理疑问和初步解法后，直接找你信得过的师友或专家请教；朋友圈内没有，就去[在行](https://www.zaih.com/)上看看有哪些行家可能带来灵感
2. 把需求抛出到朋友圈或你参加的一些专题群里，撬动更多力量
3. 如果是想检验一些自己的设想是否靠谱，可以试试问问主理人，比如 ta 有经历过类似的情况吗，当时是怎么解决的，如果重来一遍会怎么处理等

Tips：

如果你想在需要时更易撬动外力支持，不妨日常就留心积累信任。比如多多分享感触启发、表达欣赏和感谢（例会互动环节也是个特别好的机会）等。此外，正确求助并及时和对方分享后续进展，也是增进信任的好办法：）






### 4. 如何形成适合自己的解法？

和做人生战略规划的方法类似：调研相关最佳实践了解实现对应目标的关键成功要素、不同阶段的重点、难点、易错点后，进行 SWOT 分析因地制宜，形成适合自己的策略。

具体可参考[实践攻略：借鉴最佳实践，SWOT 因地制宜 – 闪闪人生](/project-meta/p2)。

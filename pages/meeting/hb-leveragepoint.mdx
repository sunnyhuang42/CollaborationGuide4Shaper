# 全局规划会指南：激发照见四步走



- **本指南适用场景**：「闪闪人生推进器」全局规划会，2 人一组交流环节

- **交流目的**：

  1. 激发主创者迭代出更靠谱的人生规划和下一周期的目标计划，尽早纠偏

  2. 促进彼此内化这些思考和实践框架，更易多快好省创造价值

     <br />

- Tips：小组交流分 2 轮，总计 50 分钟

  - 每轮交流一位主创者的内容，之后互换角色
  - **一轮交流 24 分钟左右，双方别忘了相互提醒时间，避免压缩下一位伙伴的时长哦**


## 流程A：适用于主创者用流程A梳理下步计划的情况

如果主创者的下步规划是用[人生规划模板](https://sunnylife.feishu.cn/wiki/wikcn5GTqumDqzAR5mwnNeYVpZf)里的【🌱流程A】梳理的，请用这版流程。否则请用流程B。

### 步骤1. 主创者分享自评得分和下步探索计划 - 3'

主创者口头分享《人生规划和下步计划模板》中模块二的下述思考：

1. **在四大维度的自评得分**。此环节仅说分值即可，先不展开分享原因，稍后再根据共创者提问来分享
2. **接下来的2周的探索计划**：
    - A3. **优先级和排期（to-test list）**：接下来2周，你打算重点探索解决哪些高风险之处，优先级是怎样的？大概怎么解决？预计需要多长时间，打算安排在什么时候进行？
    - A4. **MVP**：如果未来1周只能投入3小时，最值得先探索解决什么问题，或产出怎样的成果？怎样算是搞定了？
    - A5. **风险和预案**：假设目标没如期实现，最可能因为哪些原因？打算做哪些防范措施和应急预案？



<br />

### 步骤2. 共创者提问激发，助力找准突破口 -  10'

- 2.1 **确保主创者内容完整**：主创者分享的内容，覆盖上述模块的子问题了吗？

    如有遗漏子问题，请及时邀请主创者现场补充该问题的思考

    <br />

    <br />



- 2.2 **选择2个评分维度，追问主创者分享具体评分原因**，助 ta 更好地自我觉察、迭代行动计划：

    | 评分维度          | 主创者自评 | 共创者追问                                                         | 
    |-------------|-------|------------------------------------------------------------| 
    | 1. 激发潜能/意义感 | >= 4分 | 为什么自评4(或5)分？可否分享一下，具体是哪些地方，让你感到很为自己欣喜、自豪，或感到此生过得很有意义？            | 
    |             | < 4分  | 为什么自评3(或2、1、0)分？可否分享一下，一般做成怎样的事情，会让你感觉欣喜、自豪，或感到那段时间过得很有意义？ | 
    | 2. 顺应大势/规律  | >= 4分 | 为什么自评4(或5)分？可否分享一下，具体是哪些地方，顺应了趋势以及世界运转的规律？                       | 
    |             | < 4分  | 为什么自评3(或2、1、0)分？可否分享一下，具体是哪些地方，让你感觉还不够顺应趋势或世界运转的规律？          | 
    | 3. 发挥优势     | >= 4分 | 为什么自评4(或5)分？可否分享一下，具体是哪些地方，发挥了你的绝对优势或相对优势？                       | 
    |             | < 4分  | 为什么自评3(或2、1、0)分？可否分享一下，具体是哪些地方，让你感觉还不够发挥了你的绝对优势或相对优势？         | 
    | 4. 难度合理     | >= 4分 | 为什么自评4(或5)分？可否分享一下，具体是哪些地方，让你感觉还没有十足把握？是哪些地方让你感到跃跃欲试？            | 
    |             | < 4分  | 为什么自评3(或2、1、0)分？可否分享一下，具体是哪些地方，让你感觉还不是很有信心，或还不乐意去尝试？             | 



<br />



- 2.3 **助力主创者觉察并落实**。认真听完主创者的思考分享后，追问 ta ：

    1. 口头分享这些思考后，你有哪些新的觉察和发现？
        - 比如怎么更好地发挥自己的潜能和优势？
        - 或怎么更好地顺应世界趋势或运转的规律？
        - 或者发现了哪些拿不准、没有考虑到的地方？
    2. 你觉得未来1个月的目标规划，有什么可以借鉴的吗？
    3. 具体怎么落实呢？比如安排在什么时候进行？

<br />

### 步骤3. 共创者提问激发，降低执行风险 -8'

- 3.1 **选择思想实验**（AB二选一 + CD二选一），邀请主创者回答，助 ta 更好地自我觉察、迭代行动计划：

    | 序号 | 思想实验                                                                          | 思考提示                      | 
    |----|-------------------------------------------------------------------------------|---------------------------| 
    | A | 接下来如果其他都保持现状，先改变哪一点/解决哪个问题，能对你在未来2个月内迭代出自评达到 4 分的人生设想，带来最大帮助？                 | 比如做哪件事后，能让其他事情变得更简单或不再必要？ | 
    | B | 如果想更快迭代出你自评达到4分的人生设想，在未来一个月里，很重要但解决起来又最简单的问题是什么？                              | /                         | 
    | C | 你最欣赏敬佩的师友或 role model，如果面对和你类似的处境，ta 未来2周会怎么去探索突破那些问题？和你目前设想的方式会有哪些区别？        | 如果有需要，之后可以去请教一下对方了解更多视角哦  | 
    | D | 如果要更高效地搞定未来2周你想探索突破的那些问题，哪些人可能带来帮助？比如谁可能有相关经验，谁可能激发灵感，谁可能带来资源？你打算何时、如何获得这些支持？ | /                         | 


    <br />


- 3.2 **助力主创者落实**。认真听完主创者的思想实验结果及更多触动后，追问 ta ：

    1. 你觉得下周目标规划，有什么可以借鉴的吗？

    2. 具体怎么落实呢？比如安排在什么时候进行？

       <br />

- 3.3  **（选做）分享倾听发现**：如果还有时间，共创者可分享自己的倾听发现，促进彼此觉察。

    分享角度参考：

    1. 主创者的人生规划、自评和下步计划，令我印象最深的是什么？由此，我打算开始、停止或继续保持什么 行动或思考方式？
    2. 主创者有哪些重复出现的模式，或让我特别欣赏的特质？
    3. 对梳理目标规划计划的方法，我有哪些新的理解和感触？

    <br />



### 步骤4. 主创者小结 - 3'

- 4.1 **说说目前有哪些触动或启发**？

  比如当下的心情是怎样的？打算开始、停止或继续保持什么行动或思考方式？

  

<br />

- 4.2 **小结一下最新版的探索计划**？（如果无更新可跳过）
  - **to-test list**：经过这一轮互动，未来2周打算重点探索解决什么问题，或检测哪些设想？预计需要多长时间？
  - **MVP**：如果未来1周只能投入 3 小时，最值得先探索解决什么问题，或产出怎样的成果？怎样算是搞定了？
  - **风险和预案**：假设目标没如期实现，最可能因为哪些原因？打算做哪些防范措施和应急预案？
